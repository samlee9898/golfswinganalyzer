Tech Stack I'm thinking:

Server - Express
Database - MySQL
ORM - Prisma
AI - OpenCV, Pytorch
Cloud - AWS? Vercel? Render?


To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

This project was created using `bun init` in bun v1.3.10. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
